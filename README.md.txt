# My Advanced Blog

A modern and responsive blog created using **HTML**,and **CSS**. This project is built to practice front-end design and layout techniques.

## Features

- Hero section with call-to-action buttons and animated elements  
- Service categories section  
- Top-selling destinations cards  
- 3-step booking process  
- Font Awesome icons  
- Google Fonts integration  
- Clean and modern design

## Tools & Technologies

- HTML5  
- CSS3  
- Google Fonts  
- Font Awesome  
- Figma (Design Reference)


## Project Structure

my-advanced-blog/

├── CSS/
│   └── style.css
├── img/

└── README.md


## Setup and Usage

1. **Clone the repository**
2. **Open the project**
- Open `index.html` in your browser to view the site
- For development, use the unmagnified CSS version in `css/style.css`
- For production, the minified version `css/style.min.css` is referenced in the HTML
